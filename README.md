# OpenAPI JSON Merger

Production-quality full-stack app for merging multiple OpenAPI JSON specs into one valid OpenAPI 3.x document.

The tool can be found at [tools.ptrmc.net](https://tools.ptrmc.net)

## Tech Stack

- Frontend: React + TypeScript + Vite
- Backend: Node.js + Express + TypeScript
- Shared validation and contracts: Zod in shared module
- Upload handling: Multer (memory storage)
- JSON preview: react-syntax-highlighter

## Deployment

There is a example docker compose file.
